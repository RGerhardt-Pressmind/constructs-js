construct.js Example

With Construct random content can be generated quickly and in large quantities. Whether via JavaScript or with NodeJS.

Automatically generate a random first and last name


<script src="dist/construct.min.js" type="text/javascript"></script>

var first_name        =   construct.ug.get('first_name'); // Output: Hanna
var sur_name          =   construct.ug.get('sur_name'); // Output: Maxel
